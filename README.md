# Kriterien
* Meiste Schichten/Lagen
* Energieinhalt pro Stück
* Energieinhalt der gesamten Mehlspeise
* Größte Menge/Anzahl von Zutaten
* Allergene
* Zubereitungszeit
* Zubereitungstemperatur
* Früheste schriftliche Erwähnung
